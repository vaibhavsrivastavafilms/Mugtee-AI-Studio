# Play Store — Screenshots checklist

Capture on a **1080 × 1920** or **1080 × 2400** phone (or emulator). Use production or staging with realistic demo content.

## Required phone screenshots (min 2, max 8)

- [ ] Home / landing — hero + “Start creating”
- [ ] Quick Cut — idea input / generation in progress
- [ ] Storyboard / scenes grid
- [ ] Export / download panel
- [ ] Projects library
- [ ] Settings or account (optional)

## Tablet (optional)

- [ ] 7" and 10" if targeting tablets — same flows, landscape OK

## Feature graphic

- [ ] **1024 × 500** feature graphic (see `FEATURE_GRAPHIC.md`)
- No text smaller than ~24px at full size; brand gold on `#0a0a0a`

## Promo video (optional)

- [ ] 30s screen recording: idea → script → storyboard → export

## Before upload

- [ ] Remove debug banners, test emails, and “beta” watermarks unless intentional
- [ ] Status bar clean (dark theme, full signal mock in emulator)
- [ ] URLs in WebView builds should not show browser chrome (verify asset links / Capacitor)
